# Tech Conference 2025 Schedule

A conference schedule table displaying sessions across multiple tracks and time slots.

## Features

- Multi-track schedule layout with three parallel tracks
- Time-based organization of sessions
- Column spanning for breaks and lunch periods
- Semantic table structure with proper scope attributes for accessibility

## Technologies

- HTML5
- Table elements with accessibility features

## Schedule Overview

- **Track A** - Advanced topics (AR/VR, Voice UI, Accessibility)
- **Track B** - Web development fundamentals
- **Track C** - Design and UX topics
- Time slots from 9:00 AM to 2:00 PM with breaks
