# Hotel Feedback Form

A comprehensive feedback form for hotel guests to provide reviews and ratings about their stay.

## Features

- Personal information collection (name, email, age)
- Radio buttons for first-time visitor status
- Multiple checkbox selections for booking reasons
- Dropdown menus for service and food ratings
- Text area for additional comments
- Form validation with required fields
- Organized sections using fieldsets

## Technologies

- HTML5
- Form elements and validation

## Form Sections

- **Personal Information** - Name, email, and optional age
- **First Visit** - Radio buttons to indicate if it's their first stay
- **Booking Reasons** - Checkboxes for multiple selection options
- **Ratings** - Dropdown menus for service and food quality
- **Comments** - Text area for additional feedback
